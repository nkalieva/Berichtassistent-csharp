# Berichtsassistent 2

Excel Add-in with user-defined functions (UDFs) for automated data retrieval from a REST API backend.

## Features
- Custom Excel functions for real-time data access
- Automatic report generation and distribution
- Integration with MS SQL Server via REST API

## Technologies
- C# / .NET
- Excel-DNA (UDF framework)
- REST API integration
- MS SQL Server

## Structure
- `Daten/` – Data layer and Excel UDF functions
- `Ribbon/` – Excel Ribbon Add-in UI
